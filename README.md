## Functions Reference

### `Library:CreateWindow`
Initializes the main graphical interface.
* **Arguments:**
    * `title` (string): The text displayed at the top of the window.
* **Returns:** `Window` object.

### `Window:CreateTab`
Creates a new navigation section within the window.
* **Arguments:**
    * `name` (string): The label for the tab button.
* **Returns:** `Tab` object.

### `Tab:CreateButton`
Adds a clickable element to the tab.
* **Arguments:**
    * `text` (string): The label displayed on the button.
    * `callback` (function): The code that executes when the button is clicked.

### `Tab:CreateSwitch`
Adds a toggle element with "ON/OFF" states.
* **Arguments:**
    * `text` (string): The label for the switch.
    * `default` (boolean): The initial state (true for ON, false for OFF).
    * `callback` (function): Returns the current state (`bool`) when toggled.

### `Tab:CreateSlider`
Adds a horizontal slider for numerical input.
* **Arguments:**
    * `text` (string): The label for the slider.
    * `min` (number): The minimum possible value.
    * `max` (number): The maximum possible value.
    * `default` (number): The starting value.
    * `callback` (function): Returns the selected value (`number`) when changed.

### `Tab:CreateDropdown`
Adds a selectable list of options.
* **Arguments:**
    * `text` (string): The label for the dropdown.
    * `list` (table): An array of strings representing the options.
    * `callback` (function): Returns the selected option (`string`) when picked.

## Basic Implementation

```lua
local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/snuggg3/nexus-ui/refs/heads/main/source.luau"))()
local Window = Library:CreateWindow("Project Nexus")

local Tab = Window:CreateTab("Settings")

Tab:CreateButton("Print Hello", function()
    print("Hello World")
end)

Tab:CreateSwitch("God Mode", false, function(state)
    print("God Mode is:", state)
end)

Tab:CreateSlider("WalkSpeed", 16, 100, 16, function(value)
    game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = value
end)

Tab:CreateDropdown("Select Team", {"Raiders", "Defenders"}, function(selection)
    print("Joined:", selection)
end)
```

The UI's ScreenGui instance can be accessed by using 
```lua
WindowObject.MainGui
```

## Plugs

You can load Nexus's preset tabs or your custom ones into your UI using plugs.

```lua
local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/snuggg3/nexus-ui/refs/heads/main/source.luau"))()

local Window = Library:CreateWindow("Project Nexus")

Window.MainGui.Destroying:Connect(function()
    shared.NexusLib = nil
end)

local plugs = {
    "https://raw.githubusercontent.com/snuggg3/nexus-ui/refs/heads/main/plugs/util.luau",
    "https://raw.githubusercontent.com/snuggg3/nexus-ui/refs/heads/main/plugs/player.luau"
}

for _, url in pairs(plugs) do
    task.spawn(function()
        pcall(function()
            loadstring(game:HttpGet(url))()
        end)
    end)
end
```
